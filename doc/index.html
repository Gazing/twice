GET api/login/

Returns a list of courses.

URL

https://gradutree.me/api/login

Body

username (required)
The user's username

password (required)
The user's password

Example

curl -H "Content-type: application/json" -X POST https://gradutree.me/api/login -d {"username":"user", "password": "pass"}

Return object

username
The username of the sucessfully logged in user
