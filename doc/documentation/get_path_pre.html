GET api/path/:start/pre

Returns an object that contains the course name and code of the start course, as well as its prerequisites.

URL

https://gradutree.me/api/path/:start/pre

Parameters

start (required)
The course code of the course which you want to generate prereqs

Example

curl https://gradutree.me/api/path/CSCC09H3/pre

Return object

courseId
The course code of the queried course.

postreq
The array of courses that are dependant on this course as a prerequisite. Each of these course objects have a property for their own postreqs.
