POST api/course/:code/vote/:direction

Updates the courses collection to add the current user to the liked or disliked array depending on input

URL

https://gradutree.me/api/course/:code/vote/:direction

Parameters

code (required)
The string of the course you are trying to find.

direction (required)
Add user to liked (1), add user to disliked (-1), remove user from either array (0)

Example

curl https://gradutree.me/api/course/CSCC09H3/vote/1

Return object

empty
