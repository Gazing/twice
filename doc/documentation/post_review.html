POST api/review

Add a new review to the "reviews" colletion in the database

URL

https://gradutree.me/api/review

Body

code (required)
The course code of the course the user is reviewing.

content (required)
The content of the message for the new review.

Example

curl -H "Content-type: application/json" -X POST https://gradutree.me/api/review -d {"code": "CSCC09H3", "content": "Great course!"}

Return object

id
The unique id of the review in the database.
