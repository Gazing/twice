GET api/user/:username/info

Return user information iff the current user is the same as :username.

URL

https://gradutree.me/api/user/:username/info

Parameters

username (required)
The username which we want to get information.

Example

curl https://gradutree.me/api/user/newuser/info

Return object

username
The user's username.

program
The program that the user is enrolled in.

spec
The post of the user (minor/major/name of their specialist stream).

taken
The array of course codes of courses that the user has taken.

allCourses
The array of course codes of courses that the user has taken and wants to take.
