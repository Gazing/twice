GET api/courses/query

Returns an array of Course objects that contain the query string

URL

https://gradutree.me/api/courses/query

Parameters

code (required)
The string of the course you are trying to find.

Example

curl https://gradutree.me/api/courses/query?code=CSCC09

Return object

_id
Unique id for the course object in the database

code
The course code

prerequisites
The course prerequisites, as written on the school web page

name
The name of the course

level
The course level (100/200/300/400)

description
A description of the course

department
The course's department

campus
The University of Toronto campus where the course can be taken

term
The term that this course is available

liked
The number of people who like this course

disliked
The number of people who disliked this course

prereq
An array of codes codes that are prerequisites for this course

postreq
An array of codes codes that are dependant on this course

takenCount
The number of people who have taken this course

commentCount
The number of reviews for this course
