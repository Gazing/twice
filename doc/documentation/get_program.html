GET api/programs/:name

Return the array of the different requirements that the user must complete to gradutate with the queried post.

URL

https://gradutree.me/api/programs/:name

Parameters

post (required)
The subject post of the user (minor/major/specialist)

spec (optional)
If specialist is selected then it is required to select the specific specialist stream.

Example

curl https://gradutree.me/api/programs/ComputerScience?post=major

Return object

credits
The number of credits that must be taken for this requirement.

courses
The list of course codes of courses that can be taken to fufill the requirement.
