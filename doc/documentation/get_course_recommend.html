GET api/courses/:code/recommend

Returns an object containing the percentage of people who took other courses, in addition to the input course

URL

https://gradutree.me/api/courses/:code/recommend

Parameters

code (required)
The string of the course you are trying to find.

Example

curl https://gradutree.me/api/courses/CSCC09H3/recommend

Return object

:courseCode: Integer
The percentage of people who have also taken this course and :courseCode
