GET api/review/:id/vote/:direction

Updates the reviews collection to add the current user to the up or down array depending on input.

URL

https://gradutree.me/api/review/:id/vote/:direction

Parameters

id (required)
The id of the review which the user is placing a vote.

direction (required)
User votes for "Helpful" (1). User votes for "Not Helpful" (-1). User removes their vote (0)

Example

curl https://gradutree.me/api/review/1e3h3h3hf34/vote/1

Return object

empty
